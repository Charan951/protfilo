# 🌐 Personal Portfolio Website

This personal portfolio was built with modern tech, clean architecture, and minimal emotional damage from debugging. This is built to showcase my software development skills, projects, and technical expertise.


## 🚀 Overview

This project is fully responsive. Developed from scratch using a hybrid modern web stack. It highlights my projects, experience, technical skills, and development philosophy.


## ✨ Key Features

### Multi‑Page Architecture

* **Home Section** — A quick identity snapshot of who I am and what I build.
* **About Section** — My journey, technical stack, and the mindset I use to build.
* **Experience Section** — Real-world roles, responsibilities, and contributions.
* **Showcase Section** — Highlighted projects with detailed descriptions and tech stack
* **Contact Section** — Direct communication and professional links

### Interactive and Modern UI

* Smooth animations and micro‑interactions
* Custom hover and scroll‑based effects
* Responsive layout optimized for all devices
* Creative but performance‑optimized visual elements

### GitHub Integration and Visualization

* GitHub activity and project data visualization
* Demonstrates collaboration, consistency, and development activity

### 🧱 Clean and Scalable Architecture

* Modular and maintainable code structure
* Separation of concerns between UI, logic, and styling
* Built with scalability and future extensibility in mind
